# 🌾 AgriAssist – HEC Generative AI Hackathon 2026

AgriAssist is an AI-powered smart agriculture assistant built for the **HEC Generative AI Hackathon 2026**.  
The project aims to support farmers by providing intelligent guidance, crop insights, and agricultural decision-making support using modern AI technologies.

---

## 🚀 Project Overview

Agriculture is the backbone of many economies, but farmers often face challenges like:
- Lack of real-time crop information
- Weather uncertainty
- Pest and disease detection issues
- Limited access to expert advice

**AgriAssist** solves these problems by providing an AI-driven digital assistant for farmers.

---

## 🧠 Key Features

- 🌱 Crop recommendation support  
- 🌦️ Weather-based farming suggestions  
- 🐛 Pest and disease guidance (AI-based insights)  
- 📊 Smart agricultural decision support  
- 💬 Interactive assistant interface
